## Purpose

To allow users to create feedback form for events and reward attendees with voucher/coupons after completion of feedback

## Functions

1. User can enable / disable submission of feedback as and when they like.
2. User will be able to match the voucher with the attendees

## why?
1. Ensure that only those attended the event can do the feedback
2. Allow all attendees to easily get their voucher after the feedback (most people don't do feedback)
